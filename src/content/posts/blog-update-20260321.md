# 博客近期更新汇总

大家好！最近对我的博客进行了一系列的改进和优化，今天就和大家分享一下这些变化。

## 🎨 视觉体验升级

### 自定义字体

博客现在使用了**鸿雷小纸条青春体**作为全站字体，这是一款清新可爱的手写风格字体，让阅读体验更加温馨舒适。

字体通过 CDN 加载，确保了加载速度：

```css
@font-face {
  font-family: 'CustomFont';
  src: url('https://f.xxu6.top/2427/font.ttf') format('truetype');
}
```

### 网站图标更新

网站 favicon 已更新为个性化图标，在浏览器标签页和书签中都能显示。

## 💬 评论系统优化

### Extalk 评论嵌入

博客集成了 **Extalk** 评论系统，这是一个基于 Cloudflare Workers 和 D1 数据库构建的高性能评论系统。

**特点：**
- 🚀 边缘计算加速，首屏加载仅需 40ms
- ⚡ 查询性能提升 73%
- 💰 零成本运行（Cloudflare 免费额度内）
- 🌍 全球 275+ 节点加速

现在你可以在每篇文章底部看到评论框，欢迎留言交流！

### 浏览量与点赞统计

每篇文章都显示了浏览量和点赞数，让你了解文章的受欢迎程度。统计数据会在页面切换时自动更新，无需刷新页面。

## 🔍 SEO 优化

### 结构化数据增强

为所有页面添加了完整的 JSON-LD 结构化数据：

- **首页**：WebSite 类型
- **文章页**：BlogPosting 类型（包含作者、发布时间、修改时间、封面图等）
- **关于页**：AboutPage 类型

### Meta 标签优化

- 添加了 `canonical` 链接，避免重复内容问题
- 完善了 `robots` meta 标签，优化搜索引擎索引
- 改进了 Open Graph 标签，提升社交媒体分享效果

## 🎯 功能改进

### 打字机效果 Banner

首页 Banner 增加了打字机动画效果，副标题会轮播显示：
- "HI I am UpXuu"
- "upxu and you..."
- "Life, coding and everything"

动画流畅自然，为博客增添了一份动感。

### 页面切换优化

使用 Swup 实现平滑的页面切换效果，配合 CSS 过渡动画，带来类似 SPA 的流畅体验。

## 🧹 代码清理

### 移除统计代码

移除了 51LA 统计代码，保护用户隐私，同时减少了页面加载时间。

### 性能优化

- 优化了 CSS 和 JavaScript 的加载顺序
- 使用 `font-display: swap` 避免字体加载时的闪烁
- 图片资源使用 CDN 加速

## 📱 响应式设计

所有更新都确保了在移动端和桌面端的良好体验：
- 移动端字体大小自适应
- Banner 高度根据屏幕调整
- 评论系统完美适配各种屏幕尺寸
